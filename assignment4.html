Explanation:

- **Universal Selector (*): Applies margin, padding, and box-sizing to all elements.

- **Type Selector (p): Makes all paragraphs blue.

- **Class Selector (.special): Makes elements with class "special" bold.

- **ID Selector (#unique): Makes the element with the ID "unique" italic.

- **Descendant Selector (div p): Underlines paragraphs within divs.

- **Child Selector (ul > li): Removes list styles from list items directly inside unordered lists.

- **Adjacent Sibling Selector (h2 + p): Increases the font size of paragraphs immediately following h2 headings.

- **Attribute Selector (input[type="text"]): Adds a gray border to text input fields.

- **Pseudo-class Selector (a:hover): Changes the color of links when hovered over.

- **Pseudo-element Selector (p::first-line): Makes the first line of paragraphs uppercase and larger font size.
